Week1

1.建立一個新的倉儲，名為(40623242).github.io

2.從https://github.com/mdecourse 中clone 2018fall 

git clone https://github.com/mdecourse/2018fall 

3.將40623242.github.io clone下來

git clone https://github.com/40623242/40623242.github.io

4.將2018fall資料夾複製裡面的檔案，除了.git和README.me，然後貼到自己的資料夾

5.先用git status查看狀態>git add .增加>git commit -m "名稱" 提交>git push 推送

6.開啟python wsgi.py ，進入近端(localhost:8443)，更改標題名稱40623242

7.重複git status > git add . > git commit -m "title name" >git push ，完成
